Detailed Description

A KeplerKinematicProfilingAlgorithm is a class that houses the Kinematic Profile Completion Algorithm. This algorithm uses the spatial projection of the satellite's orbit, as obtained from the OrbitDeterminationAlgorithm, and matches it to a time-based function predicting the position of the satellite at any time. It uses Kepler's Second Law to accurately obtain the velocity profile of the satellite
